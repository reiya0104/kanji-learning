# PRD 001: 漢字読み学習アプリ

## ステータス
Draft

## 目的

iPad / スマホで毎日1〜3分触れる、自分用の漢字読み学習アプリ。
学習 7 : ゲーム 3 のバランスで、毎日継続して苦手漢字を潰す。

## コアループ

1. 10問セッションで読み問題（4択）を解く
2. 間違えた問題を記録する
3. 次回セッションで再出題する

## 問題形式

- 漢字（または熟語）を表示する
- 読み仮名の4択から選ぶ
- 制限時間なし（集中モードは将来検討）

## 復習ルール

- 初回ミス → 当日セッション内で再出題する
- 2回連続正解 → 苦手フラグを解除する

詳細：[docs/adr/0003-review-rule.md](../adr/0003-review-rule.md)

## 非機能要件

- オフライン動作（ローカル保存）
- iPad / iPhone 両対応
- 起動からセッション開始まで3秒以内
