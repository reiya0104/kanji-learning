# ADR 0004: データ保存戦略

## ステータス
Accepted

## コンテキスト
学習記録（ReviewRecord）をどこに保存するかを決める必要がある。
個人用アプリのため、初期はシンプルさを優先する。

## 決定
初期はデバイスローカル保存に限定する。

## 理由
- 外部サーバーが不要で、インフラコストがかからない
- オフライン動作が保証される
- 個人利用のため、データ損失リスクは許容範囲内

## トレードオフ
- 機種変更時にデータが失われるリスクがある（バックアップ機能は将来検討）
- 同期機能は v0.3 以降の検討事項とする
- 将来的には iCloud または独自バックエンドへの移行を検討する
