# kanji-learning

iPad / スマホで毎日1〜3分触れる、自分用の漢字読み学習アプリ。

## 概要

学習 7 : ゲーム 3 のバランスで、毎日継続して苦手漢字を潰すことを目標とする。

**コアループ**
1. 10問セッションで読み問題（4択）を解く
2. 間違えた問題を記録する
3. 次回セッションで再出題する
